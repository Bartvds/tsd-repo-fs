## Build

Install development dependencies in your git checkout:

````bash
$ npm install
````

Build and run tests using [grunt](http://gruntjs.com):

````bash
$ grunt build
````

````bash
$ grunt test
````

````bash
$ grunt clean
````

See the `Gruntfile.js` for additional commands.
